Game Instructions

Welcome to CodeQuest Adventure! Here's how you can play the game:

  1. Select a level from the levels page.
  2. Read the challenge description and understand the task.
  3. Write your code in the code editor provided.
  4. Click on the "Run Code" button to execute your code.
  5. If you're stuck, click on the "Hint" button to get a hint.
  6. Once you solve the challenge, proceed to the next level.

Good luck and have fun learning!
